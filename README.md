## For Aws-cli Download
- Install aws-cli
```sh
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```
- Set aws configurations
```sh
aws configure
```
Then enter the AWS ID and AWS Key.
